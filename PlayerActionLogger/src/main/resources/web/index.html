EVIDENCE MODEL

Readable text logs remain primary

Normal actions stay in logs/players/<username>.log. UUID history stays in logs/players/_uuid-index.csv. PvP incident files stay in logs/combat/.

LIGHTWEIGHT DESIGN

No database or movement flood

Disk writes use a background writer thread. Dashboard summaries are kept in memory. Movement is used only for chunk-level context and is not written line by line.

IMPORTANT LIMIT

Flags are not verdicts

The plugin cannot know voice-chat warnings, permission given elsewhere, or every off-server fact. Alerts point reviewers to evidence.
