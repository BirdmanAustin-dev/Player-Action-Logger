PlayerActionLogger is a Paper Minecraft moderation plugin designed to help server owners review player activity. It creates readable username-based logs, maintains a separate UUID index, tracks chat, commands, deaths, block actions, item movement, bucket use, and key server events, and includes a local searchable web dashboard. The readable log format also works well with LLMs like GPT, Gemini, and Claude, making it easier to analyze massive amounts of player activity data for moderation, investigations, and pattern detection.
